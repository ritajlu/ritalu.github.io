Catcus.me
School Project (November 2014)
A pet ownership simulator in the form of a web app. Users can adopt different kinds of pets (such as cacti, cats) and use items (such as water, food) on them to fulfill their daily needs. The virtual pets' need levels will decrease hourly regardless of whether the user is using the site or not to give users an idea of the constant responsibility that comes with owning a real pet.
Developed using the Laravel MVC PHP framework with Shanna Wang and Lei Lopez
